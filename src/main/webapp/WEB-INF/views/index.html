iZoneとは

写真販売に特化したECサイトです。
誰でも簡単にユーザー登録して、写真を販売することができます。

今すぐ登録

写真をアップロード

  • 撮影した写真をイベントフォルダにアップロードします。
  • アップロードした写真には透かし文字が印字されます。

金額設定と公開

  • アップロードした写真の販売金額を設定します。
  • イベントフォルダの公開パスワードを設定します。

販売対象に連絡

  • 設定した公開パスワードを販売対象に連絡します。
  • パスワードの代わりに公開QRコードを使用できます。

写真の購入

  • 公開パスワードまたはQRコードでログインします。
  • 写真を選択してクレジット支払で購入します。
