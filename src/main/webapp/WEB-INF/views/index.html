iZoneとは

写真販売に特化したをECサイトです。
誰でも簡単にユーザー登録して、写真を販売することができます。

[[#{label.signup.button}]]

写真をアップロード

  1. 撮影した写真をイベントフォルダにアップロードします。
  2. アップロードした写真には透かし文字が印字されます。

金額設定と公開

  1. アップロードした写真の販売金額を設定します。
  2. イベントフォルダの公開パスワードを設定します。

購入対象に連絡

  1. 設定した公開パスワードを購入対象に連絡します。
  2. パスワードの代わりに公開QRコードを連絡することもできます。

写真の購入

  1. 公開パスワードまたはQRコードでサイトにログインします。
  2. 写真を選択してクレジット決済します。
